Geomancer: LLM Decision Making

CPSC 490 Senior Thesis — A web-based tool that enhances human decision-making through Large Language Model assistance with structured prompting and validation.

  • Python
  • OpenAI GPT-4
  • LangChain
  • Pydantic
  • Few-Shot Prompting

December 2024

Course: CPSC 490

Institution: Yale University

GitHub: github.com/btd2026/geomancer

Geomancer is a web-based tool designed to enhance human decision-making through Large Language Model (LLM) assistance. By leveraging OpenAI's GPT-4 with carefully crafted few-shot prompts and structured output validation via Pydantic, Geomancer transforms unstructured decision problems into actionable insights.

The system uses a wizard-based interface to gather context, applies LLM reasoning with chain-of-thought prompting, and presents structured recommendations with confidence scores.

Problem Statement

Complex decision-making often involves weighing multiple factors, understanding trade-offs, and predicting outcomes. Traditional decision support systems require extensive rule engineering and struggle with nuance. LLMs offer reasoning capabilities but need proper structuring to produce consistent, actionable outputs for real-world decisions.

Technical Approach

1. Structured Prompt Engineering
Geomancer uses carefully designed few-shot prompts that guide the LLM through systematic decision analysis. Each prompt includes examples of desired reasoning patterns, encouraging the model to consider multiple perspectives and weigh factors explicitly.

2. Pydantic Validation
All LLM outputs are validated against strict Pydantic schemas, ensuring structured, type-safe responses. This includes decision scores, reasoning chains, risk assessments, and action recommendations.

3. Wizard-Based Interface
A multi-step wizard collects contextual information from users—decision type, constraints, stakeholders, and time horizon—before invoking the LLM. This structured input ensures relevant, tailored recommendations.

Key Features

  • Decision Context Capture — Structured inputs for decision type, available options, constraints, and stakeholder information
  • Chain-of-Thought Reasoning — LLM reasoning is made explicit, showing the step-by-step evaluation process
  • Confidence Scoring — Each recommendation includes confidence levels and risk assessments
  • Multi-Option Comparison — Side-by-side evaluation of multiple decision alternatives

Implementation

  • Frontend: Clean HTML/CSS/JavaScript interface with multi-step wizard
  • Backend: Python with FastAPI for serving the decision engine
  • LLM Integration: OpenAI GPT-4 API with LangChain abstractions
  • Validation: Pydantic models for all LLM inputs and outputs

Results & Evaluation

The system successfully produces structured, actionable recommendations across diverse decision scenarios including career choices, product decisions, and strategic planning. Pydantic validation catches and allows retry for malformed LLM outputs, ensuring reliability. The few-shot prompting approach significantly improves reasoning quality compared to zero-shot baselines.

Conclusion

Geomancer demonstrates that LLMs can be effective decision-support tools when properly prompted and structured. The combination of few-shot learning, chain-of-thought reasoning, and rigorous output validation creates a system that enhances human decision-making while maintaining transparency and controllability.
