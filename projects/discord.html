Live Translator for Discord

Real-time translation built natively for Discord voice calls using streaming STT and inline bot integration.

  • Discord API
  • Deepgram STT
  • WebSocket

2024

View Live GitHub

Cross-language voice chats lose momentum when translation lags or lives outside Discord. This project aims to bring real-time translation natively into Discord voice channels with a focus on low latency and seamless integration.

Architecture

The system uses a streaming STT service (Deepgram) that feeds a translation worker. An inline bot posts both interim and final translations directly to the chat, keeping everything native to Discord.

  • Cross-language voice chats lose momentum when translation lags or lives outside Discord.
  • Goal: native feel, no context switching, clear in-channel visibility.
  • <250ms interim response, sub-second final translations.
  • Works inside Discord chat; keeps moderator controls intact.
  • Reconnection guards + jitter buffers smooth dropouts.
  • Rate-limit backoff and interim/final dedupe reduce chat spam.
  • Per-user language preferences; only opted-in voices get processed.
  • Overlay can be toggled per channel for high-visibility events.
  • Hardening reconnect paths, tuning translation consistency, building a room-health console.
